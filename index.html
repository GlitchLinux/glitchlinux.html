What is gLiTcH Linux?

gLiTcH Linux is a lightweight, security-focused operating system designed for privacy enthusiasts and hackers alike. Built with stealth and OPSEC in mind, it provides a robust platform for secure computing.

Features:

  • Pre-configured privacy tools
  • gLiTcH Linux is a Debian-based distro that focuses on privacy, anonymity, and security.
  • The .img file is bootable in both BIOS and UEFI.
  • It includes a Calamares installer but is also designed for live boot usage.
  • An encrypted persistent partition can be set up, giving users flexibility and security.
